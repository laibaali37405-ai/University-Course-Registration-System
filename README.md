# University Course Registration System UMLs

A robust system designed to manage student enrollments, course selections, and academic records efficiently. This repository features a comprehensive suite of 14 detailed UML diagrams that visualize the entire system.

## Project Overview
This project provides a complete technical blueprint for a University Course Registration System, covering the software development lifecycle from requirements to design.

## Features
- Student & Admin Authentication.
- Dynamic course selection and registration management.
- Complete UML modeling for architectural clarity.
